Syllabus

Module 1: Introduction to Classes and the Document Class
Using class files and object oriented programming, using the document class, and creating simple classes. Class files
Module 2: Event Listeners and Class Hierarchies
In this class we'll be looking at handling input through event listeners- how they work, and how to generate (and respond to) custom events. We'll also be looking at creating hierarchies of classes. Class files
Module 3: More on class hierarchies
In this class, we'll be continuing with the example from last week, and talking a bit about singleton classes and the drawing API. Class files
Module 4: Workshop / Project discussions
In this class, we took some time to let me touch base with each student about thier project and provide 1-on-1 guidance.
Module 5: Loading External Media
In this class, we'll look at the workflow for loading external data into Flash- finding data online, manipulating it with PHP, generating XML, and loading the result to drive a simple infographic. Class files
Module 6: Transitions and External Media
Module 7: Midterm presentations
