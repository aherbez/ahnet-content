Experimental Activity: Class 05

In this class, we'll be looking at uploading images from Flash, storing them in a database table, and retrieving them in a simple gallery.

The example below consists of two parts: one to upload images to a database table on my site, and the other to display all of the images so uploaded. They could certainly be combined in a single project- I've just separated them out for clarity.

Example 1: upload

Example 2: gallery

Flash Files

  • class09.as main document class for upload example
  • ImageUpload.as class to handle the image uploading
  • ImageGallery.as class to handle the display of images

PHP files

  • insert_image.php PHP file responsible for storing an image in the database
  • list_images.php PHP file that lists all of the images in the database
  • image_xml.php PHP file that outputs an XML document of all the images in the database
  • show_image.php PHP file that outputs a single image as a JPGE
