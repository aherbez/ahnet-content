Assignment

For this week's assignment, I'd like you each to create some script-based animation. You should create a flash file that uses a document class and at least one other class (such as the square class in the example), to create an animation. All of the logic for the animation should reside in your classes- there should be NO timeline-based animation.

When you are done, email me a link to the finished product, along with the source files you used to make it.
