Experimental Activity: Class 05

In this class, we'll be looking at object-oriented programming with PHP.

The example below loads data for recent earthquakes, with the data coming from: http://earthquake.usgs.gov/earthquakes/catalogs/index.php#csv As the user moves the mouse from left to right, various data points are shown or hidden depending on the magnitude of the associated quake.
