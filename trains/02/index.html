v.2

Broken, but with lots of progress. Working on building up a system for creating details by annotating edges. Things that should break through walls like doors and windows are "features" and run for a certain length along the perimeter of a polygon. The polygon then provides a list of edges which are in turn used to create extruded geometry.

Shown here is the result of chopping one door and one window into the base polygon. Definitely numerous problems, but it's definitely progress.
