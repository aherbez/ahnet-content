Introduction to CNC for Makers

This course serves as an introduction to the exciting world of small-scale computer numerical control (CNC) production for the hobbyist or small business owner. We'll provide an overview of what CNC is and how to create files needed to produce physical goods, whether they be one-offs for home use or fledgling businesses.

  1. Introduction / overview of CNC tools
    • what machining is
    • what CNC is
    • additive vs subtractive manufacturing
  2. Overview of 3d files
    • basis of 3d files (vertices, polygons)
    • non-manifold topology
    • need for watertight meshes
    • intro to the STL file format
    • types of 3d design tools
      • beginner-friendly
      • CAD based
      • artistic
      • other (repair, special-purpose)
  3. 3d model creation: common operations
    • primitive creation
    • adding detail (sketch on face, knife tool)
    • extruding
    • Boolean operations
    • vertex manipulation
  4. 3d file creation tools: CAD based
    • Fusion 360, FreeCAD, OpenSCAD
    • sketch-based modeling
    • the power of parametric operations
  5. 3d file creation tools: artistic
    • Maya,Max, Blender
    • importance of good topology
    • need to maintain good edge loop geo
    • ability to paint details onto the surface
    • basic flow:
      • start with a basic shape
      • add detail at the largest scale at each step
      • smooth
      • sculpt fine details
  6. 3d file creation: additional tools
    • cleanup tools (netfabb, meshlab)
    • photogrammetry tools
    • meshmixer for combining models
  7. design considerations for additive manufacturing: FDM
    • overhang angles
    • designing for your nozzle width and Z resolution
    • tolerances
    • print orientation and strength
  8. design considerations for subtractive manufacturing: laser cutting
    • cutting and etching
    • accounting for thickness of cuts
    • techniques for joining pieces together
  9. Closer look at GCODE
    • nature of gcode: ordered instructions
    • some common GCODE commands
    • how gcode gets generated
  10. Making custom CNC tools
    • basic recipe: microcontroller/single board computer plus one or more stepper motors
    • whats a stepper motor
    • examples of strange/special-purpose CNC tools
      • pancake bot
      • egg bot
      • wire bender
