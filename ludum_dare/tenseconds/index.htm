Ludum Dare: 10 Seconds

How to play:
  • Click to start the game
  • Click to start a round
  • Wait for the "FIRE!" signal
  • Click and swipe with the mouse to fire. The direction and speed with which you swipe will set the angle and force of the bullet
  • Click to start the next round
  • Once you've either won or lost, click to reset the game and start over
